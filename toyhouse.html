About This Project

Toyhouse is a character repository that allows Users to customize their profiles and character profiles using HTML and CSS. The website supports HTML, in-line styling, Boostrap, and FontAwesome icons, while offering CSS as a premium feature.

Designer and Programmer

I designed and coded some personal user profiles and character profiles from scratch, shown below. As I do not have the premium feature, my explorations use in-line styling and bootstrap.

I learned a lot through this experience and I really enjoy being able to own the entire process. I will be continously working on improving these designs as well as coming up with new ideas!

Sample1: User Profile

I modeled this user profile off of a phone screen, with the user's information listed in chat bubbles with a profile picture. I tried adding in a time and date display with custom colors, using timeanddate.com's free clock (shown on the right). Toyhou.se does not support this feature so it is not shown (image on the left).

Sample2: User Profile

I wanted a simpler User profile where I can easily display basic information and include necessary links. I used tooltipster to display image credits when a viewer hovers on the icons.

Sample3: Character Profile

The goal of this character profile is to show the character's basic information and story, as well as display pictures/art of the character. I created three main tabs: About, Story, and Relationships for this profile.

  • About consists of basic character information, commission notes, a simple character blurb and the character's likes and dislikes.
  • Story includes a section for an overview of the character's background as well as subsections for chapters of the character's life.
  • Relationships introduces the character's relations, showing some simple details about their friends/family/acquaintances, as well as a link to the character's page.
Sample4: Character Profile

Some of my characters work in the entertainment industry as singers, actors, models, etc, and I wanted to create an entertainment-company-artist-profile-esque character profile. There are 4 tabs for this profile: Home, About, Story, Relationships, indicated by their respective fontawesome icons.
