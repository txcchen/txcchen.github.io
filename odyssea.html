About This Project

This project is a website made for Odyssea, a fictional entertainment company developed for our original characters. The website includes a description of the company's mission and the actors, singers, models, and staff signed with the company.

The website was hosted on Boston University's CS server using Firezilla and serves as my first experience with web progamming and hosting.

Designer & Developer

The website is a static responsive website designed and coded from scratch. I experimented with various features such as Morphing Modal Window and Animate.css to add movement to character sprites and make the website fun to view.
