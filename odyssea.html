Custom Website

A custom website designed and coded from scratch for CS103 Introduction to Web Programming.

Role
Designer & Developer
Date
November 2021 - December 2021
Team
Tiffany C, Lesley, C
Technologies
HTML, CSS, JavaScript, Bootstrap, JS Utilities

Concept

This project is a website made for Odyssea, a fictional entertainment company developed for our original characters. The website includes a description of the company's mission and the actors, singers, models, and staff signed with the company, as well as a news page and a home page including recent discography.

Development

The requirements for the website included responsiveness. I experimented with various features such as Morphing Modal Window and Animate.css to add movement to character sprites and make the website fun to view and interact with. The website was hosted on Boston University's CS server using Firezilla and was taken down after I graduated. This project serves as my first experience with web progamming and hosting.

What I worked on

Characters Page

This page is sectioned into Musicians, Singers, Actors and Staff, with each section accessible by the navigation tabs near the top of the page.

Character Modal

When a character is clicked, a modal is opened to show the character and basic information with some animations.

About Page & Credits

This page includes a description of what the company is, the CEO statement, and mission statement. There is also a Credits page to attribute ownership of parts of the website, utilities used, and characters.
