About This Project

I wanted to make an utility app so I decided to recreate/redesign a simple calculator I coded for my CS103 Intro to Web Programming and Technologies class. This time, I wanted the calculator to be more intuitive to use and fun design-wise.

New Design [July2023]

This is my most recent calculator design.
This calculator is used in a way similar to official calculators - the user clicks on a number, the operation, and the second number (JS). I wanted the design to be fun! so I modelled it based on a typewriter, and styled the buttons with a 3D effect. I experimented with pressing the button to mimic a key being pressed as well as an animated cursor to mimic a typing effect (CSS).
